MOSFET Characterization MATLAB App



A MATLAB App Designer application for interactive MOSFET characterization and visualization.



Overview



This application provides a graphical interface where users can configure device parameters, run numerical simulations, and visualize MOSFET output characteristics. It is designed for quick experimentation and analysis of MOSFET behavior.



Features



Interactive GUI built with MATLAB App Designer



Configurable device parameters and solver settings



Visualization of MOSFET characteristic curves



Multiple plot tabs for analysis



Export plots and simulation data (CSV)



Requirements



MATLAB (R2022+ recommended)



Usage



Open the .mlapp file in MATLAB App Designer.



Run the application.



Configure device parameters and simulation settings.



Run the simulation to generate characteristic curves.



Files



MOSFET\_App.mlapp – Main MATLAB application
